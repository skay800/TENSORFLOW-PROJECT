Handwritten Text Recognition

A CSC 317 AIP Project by Group 1 Machine Learning. The model uses Convolutional Recurrent Neural Network to recognize the Handwritten line text image without pre segmentation into words or characters. Use CTC loss Function to train Input the Handwritten Text here as file upload and convert into digital text.
